Product updates

AI frameworks integrated with MongoDB

 

Build your next AI project with easy integrations between AI frameworks, LLMs, and Atlas Vector Search. Leverage partnerships with Langchain and LlamaIndex to enhance AI application development.

Laravel Herd adds native MongoDB support

 

Laravel Herd now includes native MongoDB support, simplifying the integration of MongoDB with Laravel applications for PHP developers using the official Laravel and PHP development environment on macOS and Windows.

The C# Driver version 3.0 is here!

 

The MongoDB C# Driver received its first major version update in eight years, offering significant enhancements in performance, functionality, and compatibility with newer C# and MongoDB versions.

Atlas Stream Processing adds support for Azure

 

Developers can now integrate MongoDB Atlas and Apache Kafka, handle complex data structures, and use the MongoDB Query API for processing streaming data.

New features for the Terraform Provider v1.22.0

 

The latest version of the Terraform MongoDB Atlas Provider introduces support for Atlas Flex Clusters (Preview), a parameter for enabling MongoDB Generative AI features, and key bug fixes, deprecations, and documentation updates.
